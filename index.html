For the back-end, I generally use Node, Express and Sequelize, but I recently used Firebase for a project and enjoyed it, and I'm curious about AWS, as well as other database systems (SQL and NoSQL alike). I'm quite enamored of JavaScript despite the language's flaws and consequent detractors, but I've used Elm and C++ some, and particularly liked their strict typing. (Possibly TypeScript is in my very near future.) For the front-end, I love React's hierarchical one-way data flow an awful lot, and I'm currently exploring React Native, but I'd love an opportunity to work with an Angular team, or possibly try one of the newer, lighter React-style frameworks. I'm a strong believer in state management systems, and Redux's ideas of centrality and immutability have greatly increased my projects' ease of scalability. But the most effective tool I've learned since starting this career is the resource that is my fellow coders. I love pair programming and learning from others' code, and trying to write code that will be a joy for them to read.
